# Devices
## Device Information

List all PCI devices and devices attached to PCI buses: 
```sh
lspci
```

DMI table decoder
```sh
dmidecode
```

## Disk Info and Partitioning
Command-line disk partitioning:
```sh
fdisk
```

List all disk partitions:
```sh
sudo blkid
```
