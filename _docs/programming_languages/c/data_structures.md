## Enumeration

Short: enum  
An enumeration is a sorted list of items. It may also be considered a set or collection of items. Example:

```c
enum WeekDays {
    day1 = monday,
    day2 = tuesday,
    day3 = wednesday,
    day4 = thursday,
    day5 = friday,
    day6 = saturday,
    day7 = sunday
};
```
